# Architecture Assets

The primary Mermaid diagrams are embedded in the root README and the documents
under `docs/` so GitHub can render them directly.

Only public-safe conceptual diagrams belong here. Do not add exports from
production documentation, infrastructure diagrams, endpoint maps, data schemas,
sequence diagrams with internal events, or screenshots of operational tools.

Before adding an architecture asset, confirm that it communicates:

1. A responsibility or engineering trade-off
2. A deliberately simplified boundary
3. No reproducible proprietary mechanism
