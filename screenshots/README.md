## Review Checklist

- No customer names, logos, content, or identifiable environments
- No prompts, configuration values, file paths, or source code
- No endpoints, tokens, IDs, logs, or developer tools
- No internal dashboard controls or operational metadata
- No visible third-party account information
- No frame that explains a proprietary implementation mechanism
- Written permission for every visual asset and character shown
